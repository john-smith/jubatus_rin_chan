凛ちゃん分類 for Jubatus
========================

プログラミング生放送勉強会のLTで話した内容のソースです

必要な物
--------

* Jubatus
* Ruby
* R, RStudio
* mecab

インストール方法はググれば出てくるのでそちらに任せます

動かし方
--------

1. rin_chan.shを実行
2. RStudioでjubatus_rin_chan.Rprojを開いてknitrボタンを押す

1は結構時間がかかります
